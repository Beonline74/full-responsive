Qu'est ce qu'un site internet de type vitrine

Il existe quattre sortes de site internet:

- les blogs
c'est comme un journal intime, sauf que c'est mise en ligne
- les sites plaquettes
c'est l'équivalent d'un flyer ou dépliant commerciale
- les sites vitrines
c'est un support sur lequel vous informez vos futurs clients de votre activité professionnelle,
il n'est en principe pas possible d'y régler une commande
- les sites e-commerces
il s'agit d'un site qui permet de faire ces achat sur internet
